---- build
    webpack dev build 命令配置项

---- config
    webpack 环境配置项

---- src
    ---- common
        ---- css
            公用 CSS 如: reset.css
        ---- js
            公用 JS
            ecar
                常用ecar组件
        ---- sass
            基础sass
    ---- components
        项目的组件

    ---- config
        项目配置文件
        index.js 对外的入口
        interface.js   所有接口配置文件
        proxy.js        代理服务器配置

    ---- pages
        项目每个页

    ---- router
        路由配置

    ---- store
        项目状态管理

    ---- utils
        项目常用工具类JS
        ajax.js                 基础 axios 封装请求接口
        maxusdata.js       等同再vuex的状态管理工具

    main.js     主入口
